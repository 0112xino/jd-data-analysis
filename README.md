# 基于MySQL的京东消费者行为与运营策略分析
## 一、项目简介
### 1.1项目背景
随着电商市场竞争加剧，深入了解消费者行为、挖掘用户购物偏好与忠诚度成为提升用户留存、优化商品策略和店铺运营的关键。本数据集包含京东平台18万+条用户行为记录，涵盖用户信息、商品信息、店铺信息等多维数据，为构建全面的用户分析提供了基础。

### 1.2项目目标
①构建用户画像：从年龄、性别、会员等级、城市等级等维度描述用户特征，分析不同人群的购物偏好与活跃规律。

②探究商品属性影响：分析品类、品牌、新品上市时间等因素与用户购买行为的关系，挖掘热门品类与品牌格局。

③评估用户忠诚度：采用RFM模型对用户进行分层，识别重要价值用户、潜力用户及流失风险用户等。

④构建行为漏斗：量化用户从浏览、加购、下单到关注的转化路径，定位转化瓶颈。

⑤分析店铺运营指标：研究店铺评分、粉丝数、会员数与销量的相关性，对比新老店铺的销售表现，并考察类目竞争结构。

⑥输出可视化仪表板：基于分析结果，利用Tableau构建可视化看板，直观展示关键指标。
